# Vue To-Do List

A simple, personal to-do list application built with Vue.js.

## 🚀 Features

- Add, edit, and delete tasks
- Mark tasks as complete
- Persistent state with local storage
- Clean and responsive UI

## 🛠️ Tech Stack

- **Frontend:** Vue.js (Composition API)
- **Styling:** CSS
